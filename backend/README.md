# My Bank Online Banking

Tema 1: La digitalizzazione dell'impresa
Traccia 6: Sviluppo di un'applicazione full stack API-based per un'impresa del settore finanziario

## Prerequisiti

*   Python 3.13.6 (per il backend)

## Backend (FastAPI)

### Installazione Dipendenze
1. Navigare nella cartella `backend/`.
2. Creare e attivare un ambiente virtuale (es. `python -m venv venv`).
3. Installare le dipendenze: `pip install -r requirements.txt`.

### Esecuzione
1. Assicurarsi che l'ambiente virtuale sia attivo.
2. Avviare il server Uvicorn: `uvicorn main:app --reload`
3. La documentazione dell'API sarà disponibile su `http://127.0.0.1:8000/docs`.
